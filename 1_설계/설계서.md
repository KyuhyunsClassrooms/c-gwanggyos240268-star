# 1. 프로그램 설계서 (5점)

## 1. 프로젝트 주제
(예: 학생 성적 관리 프로그램)
도서 관리 시스템(책 정보를 추가-삭제-검색하는 프로그램)
## 2. 핵심 기능 목록
(구현할 기능들을 순서대로 나열하세요.)
1.  프로그램 시작 시 메뉴 출력
2.  도서 정보 추가(제목, 저자, 출판연도)
3.  도서 목록 출력
4.  도서 검색(제목)
5.  도서 삭제
6.  프로그램 종료

## 3. 데이터 구조 설계
(프로그램에 필요한 핵심 변수, 배열을 설계하세요. 필요하면 구조체(struct)를 사용할 수도 있습니다.)
